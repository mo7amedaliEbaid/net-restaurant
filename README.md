## online restaurant

A Flutter app to display meals and manage orders and sales of a restaurant.

## State Management

- Getx

## Features

- Offline app (Sqflite database),
- Cart
- Favourite List
- Responsive Ui (All mobile sizes and orientations)
- Google Pay
- Authentication
