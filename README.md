## online restaurant

A Flutter app to display meals, manage orders and sales of a restaurant.
<br />

### State Management

- Getx
<br />

### Features

- Offline app (Sqflite database),
- Cart
- Favourite List
- Responsive Ui (All mobile sizes and orientations)
- Google Pay
- Authentication
<br />
